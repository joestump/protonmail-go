# protonmail-go

A Go client for the Proton Mail API.

This package was extracted from the `protonmail/` directory of
[emersion/hydroxide](https://github.com/emersion/hydroxide) so it can be
consumed as a standalone library. The original code is MIT-licensed
and copyright © 2017 emersion; see [LICENSE](LICENSE).

## Status

Initial extraction — lift-and-shift, no API changes from the upstream
package. Refactoring, documentation, and tests will follow.

## Install

```sh
go get github.com/joestump/protonmail-go
```

## Usage

```go
import "github.com/joestump/protonmail-go"
```

The package name is `protonmail`.
